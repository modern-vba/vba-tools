# vba-devtool

```text
vba-devtool <command> [options]
```

## Commands

| Commands  | Description |
|-----------|-------------|
| `new`     | Create a workbook-backed VBA project. |
| `add`     | Copy CommonModules entries into the selected document source set. |
| `build`   | Build the selected document into bin output. |
| `test`    | Run VBA unit tests for the selected built document. |
| `publish` | Publish the selected document. |
| `update`  | Update installed CommonModules entries. |
| `export`  | Export modules from a workbook into source. |
| `doctor`  | Check project and machine prerequisites. |

## new

Create a workbook-backed VBA project.

### Usage

```text
  vba-devtool new <project-name> [--document <name>]
```

### Options

| Options             | Description |
|---------------------|-------------|
| `--document <name>` | Document name. Defaults to the project name. |

## project.json

`project.json` is the project manifest generated by `vba-devtool new`. Commands use it to resolve the project root, document definitions, selected document for document-level commands, source directory, template workbook, bin output, publish output, CommonModules repository, and command defaults.

Generated `project.json` files are written as UTF-16LE with BOM. Paths in the manifest are relative to the project root unless an absolute path is required.

Example generated by `vba-devtool new SampleProject` when `../common_modules_repo` exists:

```json
{
  "schemaVersion": 1,
  "projectName": "SampleProject",
  "primaryDocument": "SampleProject",
  "documents": {
    "SampleProject": {
      "kind": "excel",
      "sourcePath": "src/SampleProject",
      "templatePath": "src/SampleProject/SampleProject.xlsm",
      "binPath": "bin/SampleProject/SampleProject.xlsm",
      "publishPath": "publish/SampleProject/SampleProject.xlsm"
    }
  },
  "commonModulesRepository": "../common_modules_repo",
  "commandDefaults": {
    "test": {
      "format": "ndjson"
    }
  }
}
```

| Field                         | Description |
|-------------------------------|-------------|
| `schemaVersion`               | Manifest schema version. Current value is `1`. |
| `projectName`                 | Project name created by `new`. |
| `primaryDocument`             | Default document used by document-level commands when `--document` is omitted. |
| `documents`                   | Document definitions keyed by document name. |
| `kind`                        | Document kind. Currently only `excel` is supported. |
| `sourcePath`                  | DocumentSourceSet directory containing the template workbook and exported VBA source. |
| `templatePath`                | Source template workbook used by `build` and `publish`. |
| `binPath`                     | Workbook generated by `build` and used by default by `test` and `export`. |
| `publishPath`                 | Workbook generated by `publish`. |
| `commonModulesRepository`     | CommonModules repository path. This is `null` when no repository is discovered. |
| `commandDefaults.test.format` | Default test output format. The generated value is `ndjson`. |
